# VersionTrail
VersionTrail is a lightweight file versioning and backup tool that automatically tracks changes in your files and folders, creating timestamped restore points so you can easily view, recover, or roll back to previous versions anytime.

This is V2.0.0 where multiple language support feature is introduced. Currently Four languages are under testing. 

1 English
2 Tamil
3 Arabic
4 Chienese

Objective of this version is to test the complete functionality of the application after which, additional languages will be made avaliable for user to select from. 

How to run:
python main.py
